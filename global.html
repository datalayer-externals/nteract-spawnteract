computeSpawnOptionsEnv(kernelSpec, spawnOptions) → {object}

This computes the SpawnOptions environment. It merges all environment variables in a specific precedence ordering. The process itself, then the kernelspecs's env from kernel.json, and then any overrides passed to spawnteract's spawnOptions.env configuration. related issue #40: merge all env variables without replacing env by spawnOptions.env
